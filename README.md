# CustomKeyboard

In a month or two I am going to start with college, since I chose my major to be Electronics I wanted to get experience with making electronic projects, hence i decided to learn pcb designing and 3d modelling necessary parts. As my first electronic project i decided to make a simple custom mechanical keyboard with no rgb or bluetooth or any display, just typing.

The whole journey making this taught me how a keyboard works, how do we design a pcb,the difference between a pcb and a schematic, fundamentals for onshape 3d modelling, Initially i struggled with grid system in kicad and placing pcb switches, and making the case.

this project was a simple keyboard with only diodes switches and a raspberry pi pico, i didnt add anything like rgb,displays,bluetooth or rotatory encoder.
